# Contact Search Web App

## Instructions

### Setup

To install dependencies, run:
```bash
npm install
```

### Running locally

To run the web app locally, run:

``` bash
npm run dev
```

This will serve the web app with hot reload at `localhost:8080`.

### Production

For production, use:
```bash
npm run build
```
